# AI ChatBot

A Java-based AI chatbot with GUI interface and FAQ response system.

## Features

- Java Swing GUI
- Rule-based chatbot
- FAQ response system
- Real-time interaction
- Enter key support
- Clear chat button

## Technologies Used

- Java
- Swing GUI
- HashMap
- OOP Concepts

## How to Run

1. Open project in VS Code or IntelliJ
2. Run Main.java
3. Start chatting
